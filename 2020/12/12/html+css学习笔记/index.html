    在此感谢 W3school 和 黑马程序员 web前端入门基础教程 ，帮助我迈进web开发的大门。

    HTML 笔记

    基本结构

    盒子

    div是行元素，独占一行，而span则是块元素，一行可以放好几个。每个盒子相当于一个容器，可以放置其他内容，其他种类的盒子同理。

    标题

    通过 <h1><h1><h1> 到 <h6><h6><h6> 定义

    <hr /><hr\ /><hr /> ​​常用于在页面中创建水平线，用来分隔内容，所以经常和标题搭配着用

    段落

    通过 <p><p><p> 定义

    段和段是分开的，如果想多打些空行，别用 <p></p><p></p><p></p> 这种操作，用一个 <br /><br\ /><br /> 即可

    图片

    通过 <img><img><img> 定义

    该标签有两个常用属性： src 是指图片的链接，绝对相对均可，空连接就用 # 就行。alt 是替代值，当图片加载不出来的时候用这段文字挡在图片应该显示的位置上面。

    超链接

    通过 <a><a><a> 定义

    普通指向外部链接

    这是最常见的，指向一个网页链接

    另外，可以使用一个 target 属性，来决定网站的打开方式

    锚（anchor）的使用

    指向该网页内部某个位置，详情可见HTML链接。

    表格

    通过<table><table><table>定义

    <tr><tr><tr>表每行，<td><td><td>表每行上面的第几列元素。

    边框属性

    具体设置可在 css 中详解

    表格的表头

    一般来说，表格都有一个专门的表头，说明这一列的元素都是啥，我们可以用<th><th><th>代替<td><td><td>。

    列表

    无序列表

    通过<ul><ul><ul>定义

    有序列表

    通过<ol><ol><ol>定义

    每个<li><li><li>相当于一个容器（行），可以在里面放一堆东西

    自定义列表

    建议这里看黑马程序员的课程，方便理解。

    通过<dl><dl><dl> 定义

    怎么说呢，这里的<dt><dt><dt>有点表格里面表头<th><th><th>内味了
